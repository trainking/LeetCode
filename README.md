# LeetCode

此仓库是我在[LeetCode](https://leetcode.com/)上的刷题记录。这是一个代码的OJ（Online Judge）网站。提供了很
多在面试中出现的算法题，而且可以通过多种语言来实现。

## 约定

1. 每一个文件夹代表一个题目。
2. 没提的多种语言解根据后缀分别。
3. 每题的不同解法，用数字区分。
